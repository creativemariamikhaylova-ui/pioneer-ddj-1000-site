# Pioneer DDJ-1000

Сайт-обзор контроллера Pioneer DDJ-1000.

## Функции

- Переключение светлой и тёмной темы
- Слайдер с фотографиями
- Увеличение картинок при наведении
- Таймер обратного отсчёта до релиза новой модели
- Кнопка плавной прокрутки вверх
- Голосование с динамической диаграммой
- Автоматическая смена фона каждые 5 секунд

## Технологии

- HTML5
- CSS3
- JavaScript

## Установка

1. Скачайте все файлы
2. Поместите изображения в папку `images`:
   - `1.png` - для страницы характеристик
   - `2.png`, `5.png`, `6.png` - для слайдера на главной
   - `3.png` - для страницы впечатлений
3. Откройте `index.html` в браузере

'''

project/
├── index.html
├── style.css
├── script.js
└── images/
├── 1.png
├── 2.png
├── 3.png
├── 5.png
└── 6.png
'''
